In order to run the scripts, the project must 
be situated as follows...

  C:/eclipse/workspace/Vesicle-Synapsin/...


Documentation is in the 
"Vesicle-Synapsin/Documentation" directory.


To test the Vesicle-Synapsin model...

  1) Go to the "Vesicle-Synapsin" directory

  2) Edit "VS.inc" to change the model 
     parameters

  3) Edit "VS.ma" to change the size of the 
     cell-space according to the formula

  4) Run VS_run.bat to generate the 
     log file

  5) Run VS_drw.bat to generate the 
     drw file

  Note: with the default values, the 
        simulation takes roughly 2 minutes
